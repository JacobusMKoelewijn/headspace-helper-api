Headspace Helper

  • Upload all your data .txt and CoA .pdf files.
  • Click Extract data.
  • Receive a Template spreadsheet!
    General:
  • Best practice is to store all Headspace output .txt files and solvent CoA .pdf files in the same diproma folder and upload all files at once.

    • CoA (pdf) files:
  • Upload up to 12 CoA files.
  • Solvent name used for the Headspace measurement should match with the CoA file.
  • For diluent use abbreviation: NMP, DMA or DMI.
  • Heptanes is not supported n-heptane is.
  • Use standardized format for all CoA files:
    "SolventName Manufacturer Catalog# Lot# ExpirationDate(mmmYYYY) purity(00.00%).pdf".

  • Example of correct CoA file:
    "Ethanol Fisher E0650DF15 2034615 Feb2025 99.99%.pdf".

    • Data (txt) files:
  • Upload up to 12 A-files
  • Upload up to 8 B-files.
  • Upload up to 5 sample files.

  • Examples of correct sample formats:
    • XYZ12345-#-#-1_rest
    • XYZ12345678-#-#-1_rest

  • Examples of correct spiked sample formats:
    • XYZ12345-#-#-S-A1_rest
    • XYZ12345678-#-#-S-A1_rest

Upload all .txt and .pdf files.
